Core Feature
Attendance tracking
Full lifecycle attendance management with IST timezone, GPS capture, and multiple status types.

Check-in with GPS

When a staff member checks in, the app captures their browser's GPS coordinates along with a human-readable location name. All times are displayed in IST (Asia/Kolkata) in 12-hour format.

  • Browser Geolocation API captures latitude + longitude
  • Location name can be typed (e.g. "Office", "Client Site")
  • Admin sees a Google Maps link for every location
  • Late detection: after 9:30 AM IST = "Late" status
  • Half-day option available on checkout for staff/manager

Admin Visit Log

When the admin checks in, it creates a "visit" record rather than personal attendance. All active users receive a notification immediately — admin can optionally add a broadcast message.

Expense Management
Fast bill submission
Staff submit expense bills with receipt uploads. Files go directly to cloud storage — bypassing server size limits for fast, reliable uploads.

Direct Upload Architecture

Receipt files upload directly from the browser to Supabase Cloud Storage using a signed URL. The server is never in the upload path, eliminating size limits and slow transfers.

  • 3-step upload: Get URL → Upload to cloud → Save metadata
  • Progress shown: "Getting upload URL… → Uploading file… → Saving…"
  • No file size limit (bypasses Vercel 4.5 MB body limit)
  • Receipts stored permanently in Supabase Storage
  • Public URL saved for instant preview/download

Team Management
User lifecycle management
Complete control over staff accounts — from onboarding to offboarding, with safe soft-delete and full data cascade when needed.
➕

Add Users

Visual role selector with card-style UI showing Staff, Manager, Admin options with icons, descriptions, and permissions. Password set on creation.

😴

Deactivate / Reactivate

Soft-delete: mark a user as inactive. Their data is preserved. Admin can reactivate them at any time from the inactive users list.

🗑️

Permanent Delete

Hard delete with full cascade: attendance records, bills, tasks, uploads, and notifications are all removed. Requires confirmation before proceeding.

🖼️

Profile Management

Users can update their own profile, change passwords, and upload a profile picture. Avatar stored in Supabase Storage.

On desktop the sidebar collapses to icon-only mode. On mobile it hides completely and opens as a full slide-in drawer with a tap on the hamburger button.

  • Desktop: collapsible sidebar with icons + labels

  • Backdrop overlay closes drawer on tap outside
